Hard knowledge

  • Comando recomendado para iniciar proyecto con Bit: npm create bit@latest.
  • Añadir Tailwind siguiendo la documentación y registrar Tailwind en la configuración de Bit.
  • Eliminar public/assets y simplificar App si se prefiere trabajar desde la nube.
  • useState: guardar estados food, calories y entries en el componente.
  • OnChange de un input: setFood(e.target.value) o setCalories(e.target.value).
  • Convertir string a número antes de sumar: Number(calories) (mencionado como "Number").
  • Suma total con reduce: entries.reduce((acc, entry) => acc + entry.foodCalories, 0).
  • Mapear entradas para renderizar lista: entries.map((entry, i) => /* JSX */) y usar key={i}.
  • Material UI: usar componentes Card, CardContent, Button, TextField, Typography, Box.
  • Material UI: la prop sx acepta un objeto JS para estilos inline (reemplaza style).
  • TextField props útiles: label, variant outline (outline), value, onChange, fullWidth.
  • Constantes fijas en mayúsculas por convención (por ejemplo DESAYUNO, COMIDA, CENA, DAILY_GOAL = 2000).
  • useEffect para cargar localStorage al montar: usar array de dependencias [] para ejecución una sola vez.
  • Cargar desde localStorage: const saved = localStorage.getItem('caloryEntries'); if (saved) setEntries(JSON.parse(saved)).
  • Guardar en localStorage al cambiar entries: useEffect(() => { localStorage.setItem('caloryEntries', JSON.stringify(entries)); }, [entries]).
  • localStorage solo guarda strings; usar JSON.stringify y JSON.parse para convertir objetos.

Summary

Se creó un proyecto React con Bit y se integró Tailwind, optando finalmente por usar Material UI para los componentes visuales. Construyeron un componente CalorieTracker que usa useState para food, calories y entries; entries.map renderiza la lista y reduce suma las calorías. Implementaron onChange con e.target.value y convierten strings a números antes de sumar. Finalmente añadieron useEffect para cargar y guardar entries en localStorage (JSON.parse al cargar y JSON.stringify al guardar).
